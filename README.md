# İstanbul Bekliyor

İstanbul Büyükşehir Belediye Başkanı Ekrem İmamoğlu için gün sayacı kampanyası yönetim paneli.

## Hızlı Başlangıç

Bu proje Claude Code ile geliştirilmek üzere hazırlanmıştır. Terminalde:

```bash
cd istanbul-bekliyor
claude
```

Claude Code CLAUDE.md dosyasını okuyarak projenin tüm bağlamını anlayacak ve paneli inşa etmeye devam edecektir.

## Proje Yapısı

```
istanbul-bekliyor/
├── CLAUDE.md                         # Claude Code ana bağlam dosyası
├── README.md
├── docs/
│   ├── algorithm-analysis.md         # X algoritması kaynak kod analizi
│   ├── campaign-strategy.md          # Kampanya stratejisi
│   ├── brand-guidelines.md           # Görsel kimlik rehberi
│   ├── research-global-playbook.md   # Küresel sansür atlama araştırması
│   ├── research-solo-campaigns.md    # Tek kişi viral kampanya araştırması
│   ├── research-avatar-design.md     # Avatar ve marka kimliği araştırması
│   └── x-algorithm-source.zip       # X algoritması kaynak kodu
├── prompts/
│   ├── daily-prompts.md              # 14 gün Nano Banana Pro promptları
│   └── profile-banner.md             # Profil ve banner promptları
├── assets/
│   ├── banner.png                    # Mevcut banner (1500x500)
│   ├── banner-source.html            # Banner kaynak kodu
│   ├── profile-photo-reference.jpg   # Profil fotoğrafı referans
│   └── banner-nano-reference.jpg     # Banner referans
└── src/                              # Panel kaynak kodu (inşa edilecek)
```
